# Butts Studios — Website

## File Structure

```
buttsstudios/
├── index.html          # Main HTML
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # All JavaScript
├── images/             # Place your images here
│   ├── showreel-poster.jpg
│   └── og-image.jpg
├── videos/             # Place your videos here
│   ├── showreel.mp4
│   └── showreel.webm
└── README.md
```

## Getting Started

Just open `index.html` in a browser, or deploy the whole folder to any static host (Netlify, Vercel, GitHub Pages, etc.).

## Asset Placeholders

Drop your real assets into the `images/` and `videos/` folders:
- `images/showreel-poster.jpg` — thumbnail shown before the showreel plays
- `images/og-image.jpg` — social sharing preview image
- `videos/showreel.mp4` / `videos/showreel.webm` — hero showreel video
